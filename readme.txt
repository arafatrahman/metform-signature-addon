The MetForm Signature Addon allows you to add a digital signature field to MetForm, enabling you to collect legally binding e-signatures directly in your forms.

== Description ==

The MetForm Signature Addon adds a digital signature field to MetForm
== Additional Information ==
The source code for `signaturepadmin.js` can be found at:
https://github.com/szimek/signature_pad

== Installation ==

1. Download the plugin .zip file.
2. Upload it to your WordPress installation via `Plugins > Add New > Upload Plugin`.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Make sure MetForm is installed and active.

== Features ==

* Add digital signature fields to MetForm.
* Responsive and mobile-friendly signature input.
* Customizable signature field options.
* Secure and encrypted signature data.
* Easy integration with existing forms.

== Frequently Asked Questions ==

= Do I need MetForm to use this addon? =
Yes, this addon is built specifically for the MetForm plugin.

= Are the signatures legally binding? =
Digital signatures collected using this addon can be legally binding, depending on your local laws. Consult a legal professional for specific guidance.
